Migrate Kanban Projects

ℹ️ About this migration

This tool will migrate your Kanban projects from localStorage to the SQLite database. This is a one-time operation that preserves all your project data including client names, status, notes, and pinned state.

⚠️ Important

After migration, the app will use the SQLite database instead of localStorage. Make sure you have a backup of your data if needed.
